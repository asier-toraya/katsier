# Portfolio 2026

Portfolio web estático, con panel de perfil, desplegable de proyectos y writeups.

## Stack
- HTML
- CSS
- JavaScript (vanilla)

## Estructura
- `index.html`: estructura principal de la página.
- `styles.css`: estilos y responsive.
- `script.js`: lógica del desplegable.
- `images/`: assets de fondo y proyectos.

## Ejecutar en local
Abre `index.html` en el navegador.
